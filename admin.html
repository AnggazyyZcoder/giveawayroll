Database Setup

Setup JSONBin.io

⚠️ PENTING: Konfigurasi Sebelum Digunakan

1. Daftar di jsonbin.io
2. Buat Bin baru (Private)
3. Copy Bin ID dan Master Key
4. Edit file script.js bagian CONFIG
5. Ganti JSONBIN_KEY dan BIN_ID
6. Klik tombol "Inisialisasi Database" di bawah

script.js — CONFIG
const CONFIG = {
  JSONBIN_KEY: '$2a$10$YOUR_MASTER_KEY',
  BIN_ID: 'YOUR_BIN_ID',
  ...
};
Setelah inisialisasi, default user: admin / admin123
