FAQ
Bagaimana cara membeli key?
Pilih produk yang Anda inginkan, klik card produk, pilih paket (Access Full Key atau Rental), masukkan nomor WhatsApp, dan klik "Buy Now". Pastikan saldo Anda mencukupi. Admin akan memproses pesanan Anda dan mengirimkan key.
Bagaimana cara top up saldo / credit?
Hubungi admin melalui saluran WhatsApp kami untuk proses top up saldo. Admin akan mengirimkan saldo ke akun Anda setelah konfirmasi pembayaran.
Apa perbedaan Access Full Key dan Rental Key?
Access Full Key memberikan akses berdasarkan durasi hari (1 hari, 3 hari, dst). Rental Keys memberikan akses berdasarkan durasi jam (1 jam, 2 jam, dst). Pilih sesuai kebutuhan Anda.
Bagaimana cara menggunakan kode promo?
Saat memilih produk dan paket, masukkan kode promo di kolom "Kode Promo" lalu klik "Pakai". Sistem akan otomatis memvalidasi dan menerapkan diskon pada harga pembelian Anda.
Apakah key dijamin aman dari banned?
Ya, Drip Client menggunakan teknologi anti-ban terbaru. Key kami didesain untuk keamanan maksimal. Namun tetap gunakan dengan bijak sesuai petunjuk yang diberikan bersama key.
Berapa lama proses pengiriman key?
Admin memproses pesanan dalam waktu 1-30 menit tergantung jam operasional. Key akan langsung muncul di menu "Logs Transaksi" Anda setelah diapprove admin.
Apa yang harus dilakukan jika key tidak berfungsi?
Coba fitur "Reset Key" di menu Quick Action (klik ikon titik 3). Jika masih bermasalah, hubungi admin melalui saluran WhatsApp kami dengan menyertakan ID transaksi Anda.
