FAQ
Apa itu Drip Client?
Drip Client adalah platform jual beli key / lisensi untuk berbagai produk digital. Kami menyediakan akses cepat dan aman untuk semua kebutuhan kamu.
Bagaimana cara membeli?
Pastikan saldo kamu cukup, pilih produk yang diinginkan, pilih paket, masukan nomor WhatsApp, lalu klik "Buy Now". Admin akan memproses pesananmu dan mengirimkan key.
Bagaimana cara top up saldo?
Hubungi admin melalui saluran WhatsApp kami. Admin akan memproses top up saldo akun kamu setelah konfirmasi pembayaran.
Berapa lama proses pengiriman key?
Proses pengiriman key biasanya berlangsung 1-15 menit setelah pembelian dikonfirmasi. Admin aktif memantau setiap hari.
Apakah ada garansi produk?
Ya! Semua produk bergaransi sesuai durasi pembelian. Jika ada masalah, hubungi admin melalui WhatsApp dan kami akan segera membantu.
Bisakah saya menggunakan kode promo?
Ya! Kamu bisa memasukan kode promo saat checkout. Setiap kode promo hanya bisa digunakan sekali per akun dan memiliki batas pemakaian total.
